## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/aanoorgroups.git

# Navigate to project directory
cd aanoorgroups

# Install dependencies
npm install

# Start development server
npm start
```

### Build for Production

```bash
npm run build
```
